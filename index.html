Install on iPhone
Set up daily auto-wallpaper in 5 minutes.
1

Open Shortcuts

Open Shortcuts → Automations → create new automation.

2

Set daily trigger

Select Time of Day → 6:00 AM → Daily. Toggle Run Immediately.

3

Get Contents of URL

Add Get Contents of URL action and paste your KJo-fy link.

4

Set Wallpaper Photo

Add Set Wallpaper Photo. Set to lock screen only. Disable preview & crop.

5

Done!

Press play to test. Every morning, a new dialogue greets you.

1

Install MacroDroid

Download MacroDroid from the Play Store (free).

2

Create a Macro

Tap Add Macro → name it KJo-fy Daily.

3

Set Trigger

Triggers → Date/Time → 6:00 AM Daily.

4

Download Image

Actions → HTTP Request → GET your KJo-fy URL. Save to /sdcard/Pictures/kjofy.jpg.

5

Set Wallpaper

Actions → Set Wallpaper → Lock Screen → select the saved file.

6

Done!
